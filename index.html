CSS Positioning for Dummies
Position Description HTML Code Advantages Disadvantages
Static Positions an element based on the order written in the HTML page position:static Simple way to view the ordering of your HTML code Lacks customisation and is reliant on HTML
Relative Functions the same as Static, however allows you to make adjustments top to bottom and left to right position:relative Provides additional customisability versus Static in terms of positioning Elements can overflow if not positioned correctly
Absolute Removes element from document flow; other elements function as if the absolute element did not exist position:absolute Great if you want to isolate certain elements without adjusting the flow of the document Positioning consideration required when adding additional elements
Fixed Removes element from the document flow and will stay in the same position on the page irrespective of scrolling position:fixed Useful if you want to have an element always visible (e.g. Navigation Bar) Constant visibility of the element may distract users
Sticky Similar to relative positioning in how it functions. Key addition is that it will 'stick' on the page even when scrolling position:sticky Useful if you want to have an element always visible (e.g. Navigation Bar) Constant visibility of the element may distract users
